目录与路径

绝对路径与相对路径

  • 绝对路径：由根目录开始写起 /home/dengdeng/.barasd
  • 相对路径：相对于目前路径的文件名写法 如 ../dengdeng

目录的相关操作

  • . 代表次层目录
  • .. 代表上层目录
    • 代表前一个工作目录
  • ~ 代表“目前身份用户”所在的home目录
  • ~dengdeng 代表dengdeng这个用户的家目录
  • 常见的目录操作指令
    • cd：变换目录（change directory）
    • pwd：显示当前目录(print working directory)
    • mkdir：建立一个新的目录
      • mkdir denglinzhe //建立一个名为denglinzhe的文件夹
      • mkdir -p /users/denglinzhe/one/two/three //系统会依次建立文件夹 直到 three
      • mkdir -m 711 test // 赋予新建的test目录初始权限，权限为711 == drwx--x--x
    • rmdir：删除一个空的目录

关于执行文件路径的变量 $PATH

6.2 文件与目录的管理

  • 文件与目录的检视 ls
    • ls -a 全部的文件（连同隐藏文件一起显示出来）
    • ls -d 仅仅列出目录本身
    • ls -h 列出人们可以认识的容量大小 kb/gb
    • ls -t 按照时间排序
  • 复制 cp
    • cp eg:cp ~/desktop/章章.jpg /users/denglinzhe //将desktop里的章章.jpg文件移动到denglinzhe文件夹里
    • -a 常用于复制文件夹
    • -i 如果目标文件已经存在时，在覆盖时会问你是否覆盖
    • -p 连同文件的属性（权限、用户、时间）一起复制过去
  • 删除 rｍ
  • 移动 mv

获取路径的文件名与目录名称

- basename 
- dirname

6.3文件内容查看

  • cat由第一行开始显示内容
  • tac 从最后一行开始显示，可以看出tac是cat倒着写的
  • nl 显示的时候同时输出行号
  • more 一页一页的显示文件内容
  • less与more类似，但是比more更好的是，他可以往前翻页
  • tail 只看后面几行
  • od 以二进制的方式读取文件
