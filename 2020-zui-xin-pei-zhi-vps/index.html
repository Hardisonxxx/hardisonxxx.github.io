更改内核

由于我们安装速锐加速，网上现有的速锐解版是需要特定版本的。以91yun为例需要将内核换为：

CentOS6内核更换为：2.6.32-504.3.3.el6.x86_64

rpm -ivh http://soft.91yun.pw/ISO/Linux/CentOS/kernel/kernel-firmware-2.6.32-504.3.3.el6.noarch.rpm
rpm -ivh http://soft.91yun.pw/ISO/Linux/CentOS/kernel/kernel-2.6.32-504.3.3.el6.x86_64.rpm --force

CentOS7内核更换为：3.10.0-229.1.2.el7.x86_64

rpm -ivh http://soft.91yun.pw/ISO/Linux/CentOS/kernel/kernel-3.10.0-229.1.2.el7.x86_64.rpm --force

查看内核是否安装成功

rpm -qa | grep kernel

其中列表里面要出现这个内核名称（以CentOS6为例：）

重启 然后查看内核是否安装成功

reboot

uname -r

搭建SS/SSR服务

下载git

yun install git

下载ss/ssr软件包

git clone https://github.com/suniceman/ss-fly

运行ss/ssr安装包

ss-fly/ss-fly.sh -i suniceman.com 1024  #ss (安装ss时候用这个)

ss-fly/ss-fly.sh -ssr  #ssr(安装ssr用这个)

安装启用速锐加速（速锐只适用于KVM的vps）

wget -N --no-check-certificate https://github.com/91yun/serverspeeder/raw/master/serverspeeder.sh && bash serverspeeder.sh

如果报错没有wegt 那就安装一个

yun -y install wget

chattr -i /serverspeeder/etc/apx* && /serverspeeder/bin/serverSpeeder.sh uninstall -f
