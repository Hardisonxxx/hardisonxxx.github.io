deepin下安装KDE桌面环境，以及在kde中运行wine程序

2020.05.24
#KDE #Linux

用久了Linux就想折腾。所以今天突发奇想，想换个桌面玩一玩，然后选择了kde....

注意：安装有风险，折腾需谨慎

在终端中输入

sudo apt install kde-plasma-desktop

然后输入密码 静候佳音...

过一会，出来一个设置面板后选择 SDDM

再过一会。。就安装好啦

重启后左上角选择Plasma

开始kde之路

突然发现在kde环境下wine的软件都无法使用 比如wine-tim,wine-qq啊啥的， 然后搜索发现

在非gnome桌面环境下，Deepin-Wine软件大部分不能启动，例如deepin-Tim，deepin-QQ以及微信等。按照DLM的说法，深度在打包Deepin-Wine软件时加入了对Gnome的依赖。通过安装gnome-settings-daemon即可解决。

所以 在终端中输入

sudo apt install gnome-settings-daemon

以安装gnome-settings-daemon

然后

cp /etc/xdg/autostart/gnome.SettingsDaemon.XSettings.desktop ~/.config/autostart

使得gnome-settings-daemon加入开机自启菜单里

然后在 系统设置>开机和关机>自动启动 里的自启动打勾,使其保持开机自启动。然后重启...
