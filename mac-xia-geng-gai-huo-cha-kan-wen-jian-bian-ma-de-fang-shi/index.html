mac 下更改或查看文件编码的方式

2020.03.06
#vim #mac

使用vim

在vim中查看编码文件

:set fileencoding

使用Vim修改：

:set fileencoding=utf-8

使用enca

ence是在unix下的一款文件编码管理软件

enca安装：

Linux（debain系）：

sudo apt-get install enca

mac:(需要安装brew)

brew install enca

enca使用

检查文件编码

enca -L zh_CN xxx.txt  #检查文件名为xxx.txt中 中文的编码方式

转换文件编码

enca -L 当前语言 -x 目标编码 文件名

-L zh_CN 一般也可以不用指定

批量转换

enca -x 目标编码 *
