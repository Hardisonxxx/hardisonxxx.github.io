Python 试题总结

2019.10.16

二级Python

第一套题

  • 命名规则：字母数字下划线
  • Python中 \0 不代表字符串结尾 只是个简单的转义字符
  • Python是一种解释性高级通用脚本语言，可以用于几乎任何与程序设计相关应用的开发
  • 在Python中算数运算符"//"表示整数除法,返回一个整数 而"/"则单纯的表示浮点数除法,返回浮点结果
  • python中break意为结束循环,continue是结束当前循环进入下一个循环
  • python中的循环结构有for语句和while语句两种
  • range() 函数返回的是一个可迭代对象 eg:range(1,6)返回1,2,3,4,5 ; range(5)等价于range(0,5)
  • return语句用来结束函数并将程序返回到函数被调用的位置继续执行,return语句可以出现在函数中的任何部分,可以同时将多个运算结果返回给函数被调用处的变量
  • python语言通过保留字def定义函数
  • ls[2][1]指的是列表中序号为2的元素中序号为1的元素
  • d.get(key,default) 方法是用来返回key对应的值,如果字典中不存在该键,则返回default
  • 序列类型(元素之间存在先后关系,通过序号访问):字符串类型,列表类型,元组类型
  • 映射类型(字典是可变数量键值对的数据结构,通过字典类型实现映射):字典类型
  • ord()函数是把字符串转换成对应的Unicode码
  • chr()函数是ord()函数的逆运算,把Unicode码转换成对应的字符输出
  • writelines()方法是将一个元素为字符串的列表整体写入文件
  • write()方法是向文件写入一个字符串或字节流,每次写入后,将会记录一个写入指针
  • 二维列表对象输出为CSV格式文件,采用遍历循环和字符串的join()方法相结合
  • csv文件的每一行是一组数据,可以用列表,元组来表示
  • 从csv文件获得数据内容后,可以用replace()来去掉每行最后的换行符
  • 若一个列表变量里的元素都是字符串类型,则可以用join()合成字符串
  • Python文件读取方法:
    • f.read()从文件中读入整个文件的内容
    • f.readline()从文件中读入一行内容
    • f.readlines()从文件中读入所有行,以每行为元素形成一个列表
    • f.seek()改变当前文件操作指针的位置
  • Python文件写入方法:
    • f.write()
    • f.writelines()
  • 一维数据采用线性方式组织,在Python中主要采用列表形式表示
  • 集合不属于线性结构
  • 二维数据采用二维表格方式组织,在Python中可以采用二位列表形式表示
  • 高维数据由键值对类型的数据构成,采用对象方式组织,在Python中可以采用字典类型表示
  • 属于Python数据分析及可视化的第三方数据库有:numpy,scipy,pandas,matplotlib,TVTK,mayavi等等
  • 属于Python Web开发方向的Python第三方生态库有:Django,Pyramid,Flask等等

第二套题

  • python三种基本数字类型:整数类型,浮点数类型,复数类型
  • 静态语言采用编译方式执行,脚本语言采用解释方式执行
  • python基本控制逻辑:顺序结构,选择结构(分支结构),循环结构,异常处理
  • eval()函数是把字符串转换成Python可用的语言
  • python中定义类的关键词是 class
  • ls.remove(x) 是删除列表ls中出现的第一个元素x
  • append(): 连接字符串
  • split()方法通过指定分隔符对字符串进行切片,返回分割后的字符串列表
  • numpy是python的一种开源数值计算扩展的第三方库,用于处理数据类型相同的多维数组
  • PIL是python语言在图像处理方面的重要第三方库

第三套题

  • 在python中 0.1+0.2 = 0.30000000000000004
  • 在python中 字符编码以Unicode形式存储
  • complex() 方法返回复数 complex(10.99) 返回 (10.99+0j)
  • 在python语言中 使用for..in..方式循环不能遍历的类型是 : 复数
  • python中全局变量和局部变量的命名可以相同
  • 函数定义时 可选参数一般放置在非可选参数的后面
  • 列表可以使用比较操作符对列表进行比较

第四套题

  • 在Python语言中 IPO == Input Process Output
  • 复数实部可以为0
  • insert()方法是 在列表的任意位置插入元素
  • 二位数据也称表格数据 , 由关联关系数据构成

第五套题

  • 文本文件和二进制文件的存储方式不同,所以文本文件不可以用二进制文件方式读入
  • 浮点数3.0和整数3具有相同的值, 其硬件执行单元,计算机指令处理方法和数据类型均不相同
  • random.uniform(a,b):生成一个[a,b]之间的随机小数
  • reverse()方法的作用是将整个列表中的元素反转

第六套题

  • 文件的seek()方法用来定位文件的读/写位置, tell()方法用来返回文件指针的当前位置
  • 在python类的定义中,以一个下划线开头的成员是保护成员,以两个下划线开头和结束的成员是系统定义的特殊成员,以两个或多个下划线开头但不以两个或多个下划线结束的成员是私有成员
  • all(x)函数是当组合类型变量x中所有元素为真是返回True
  • spilt() 是字符串分隔函数
  • ascii()返回括号内的参数字符串
  • 类是一种抽象事物的分类 , 对象才是一种实例
  • lambda关键字是用来定义匿名函数的 格式为 <函数名> = lambda <参数列表>:<表达式>

第七套题

  • 元组与列表类似,可存储不同类型的数据.元组是不可改变的,创建后不能再做任何操作
  • 一个类继承另一个类,那么被继承的这个类称为超类或者父类
  • 如果调用的是继承父类的公有方法,可以在公有方法中访问父类的私有属性和私有方法
  • id() 是Python中的内置函数,用来查看变量在内存中的地址
